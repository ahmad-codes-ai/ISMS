# Python Practice: Inventory & Sales System

I built this project to practice core Python concepts. It focuses on using functions, managing data structures, and building a complete menu-driven program.

## Why I Built This
* To practice creating and calling **Python functions**.
* To learn how to build interactive, **menu-driven terminal programs**.
* To work with **lists and dictionaries** for data management.
* To practice basic **business logic** like calculating profits, stock updates, and product returns.

## Core Features
* **Inventory Management:** Add, view, search, update, delete, and restock products.
* **Sales Tracking:** Sell items, process customer returns, and track sales history.
* **Business Reports:** Generate automated total profit summaries and find the top-selling product.
